16. Цикл for

Перебор элементов последовательности

Синтаксис for

for перебирает элементы любой последовательности:

# Перебор строки
for char in "Python":
    print(char)  # P y t h o n

# Перебор списка
for fruit in ["яблоко", "банан", "вишня"]:
    print(fruit)

for с range()

range() генерирует последовательность чисел:

for i in range(5):
    print(i)  # 0 1 2 3 4

for i in range(2, 7):
    print(i)  # 2 3 4 5 6

for i in range(0, 10, 2):
    print(i)  # 0 2 4 6 8

for с else

else выполняется, если цикл не был прерван break:

for num in [1, 2, 3, 4, 5]:
    if num == 3:
        print("Нашли 3!")
        break
else:
    print("3 не найдено")

# Вывод: "Нашли 3!" (else не выполнился)

Переменная цикла

Переменная цикла существует и после завершения цикла:

for i in range(3):
    pass
print(i)  # 2 (последнее значение)
