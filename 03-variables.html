3. Переменные

Объявление, присваивание, соглашения об именах

Что такое переменная?

Переменная — это имя для области памяти, в которой хранятся данные. В Python переменная создаётся в момент первого присваивания: 

name = "Анна"
age = 25
pi = 3.14159

Тип переменной определяется автоматически (динамическая типизация): 

x = 10        # int
x = "hello"   # теперь str
x = 3.14      # теперь float

Правила именования

  • Имя может содержать буквы, цифры и символ подчёркивания _
  • Не может начинаться с цифры
  • Регистр имеет значение: age ≠ Age
  • Нельзя использовать ключевые слова Python (if, for, while и др.)
PEP 8 — стиль: используйте snake_case для переменных: user_name, total_count.

Множественное присваивание

a, b, c = 1, 2, 3
a = b = c = 0     # всем трём присвоится 0

Полезные функции

Функция Описание
type(x) Возвращает тип переменной
id(x) Возвращает адрес объекта в памяти
isinstance(x, type) Проверяет принадлежность типу

Базовые типы данных

В Python есть несколько встроенных типов, которые используются постоянно:

Тип Пример Описание
int 42, -5, 0 Целые числа (неограниченной длины)
float 3.14, -0.5, 1e3 Числа с плавающей точкой (дробные)
str "привет", 'a' Строки (текст)
bool True, False Логические значения (истина / ложь)
NoneType None Отсутствие значения (аналог null в других языках)
Важно: True и False пишутся с большой буквы. None — тоже с большой буквы. Это не строки, а специальные значения языка.

Преобразование типов

Python позволяет явно преобразовывать данные из одного типа в другой с помощью функций-конструкторов: 

# Преобразование в int
int(3.99)        # 3 (отбрасывает дробную часть)
int("42")        # 42 (строка → число)
int(True)        # 1
int(False)       # 0

# Преобразование в float
float(5)         # 5.0
float("3.14")    # 3.14
float("1e3")     # 1000.0

# Преобразование в str
str(42)          # "42"
str(3.14)        # "3.14"
str(True)        # "True"

# Преобразование в bool
bool(0)          # False
bool(1)          # True
bool("")         # False (пустая строка)
bool("текст")    # True
bool(None)       # False

Правила, по которым значения становятся False (все остальное — True):

  • Числовой ноль: 0, 0.0
  • Пустая строка: ""
  • Значение None
Лайфхак: bool(x) — это быстрый способ проверить, «заполнена» ли переменная. Например, bool(input("Введите имя: ")) вернёт True, если пользователь что-то ввёл.

Опасные преобразования

При попытке преобразовать неподходящую строку возникнет ошибка:

int("abc")       # ValueError
float("12,5")    # ValueError (нужна точка, а не запятая)
int("3.14")      # ValueError (строка содержит точку, int такое не умеет)
