17. Булевы переменные

True, False, bool(), truthy и falsy значения

Тип bool

bool — это подтип int. True = 1, False = 0:


is_active = True
is_finished = False

print(type(is_active))  # 
print(True + 1)         # 2
print(False * 5)        # 0

Функция bool()

bool() преобразует любое значение в True или False:


# Falsy значения (преобразуются в False):
print(bool(0))        # False
print(bool(0.0))      # False
print(bool(""))       # False
print(bool(None))     # False
print(bool([]))       # False (пустой список)
print(bool({}))       # False (пустой словарь)
print(bool(set()))    # False (пустое множество)

# Truthy значения (всё остальное):
print(bool(1))        # True
print(bool(-1))       # True
print(bool("text"))   # True
print(bool([1, 2]))   # True

Логические операции

and, or, not с булевыми значениями:


print(True and False)  # False
print(True or False)   # True
print(not True)        # False

# С truthy/falsy значениями:
print(0 and 42)        # 0 (and возвращает первое falsy)
print(3 or 0)          # 3 (or возвращает первое truthy)
print("" or "default") # "default"
