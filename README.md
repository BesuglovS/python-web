## 🚀 Как использовать

1. **Откройте** `index.html` в любом современном браузере — это главная страница с навигацией.
2. **Кликните** на любую тему, чтобы перейти к соответствующей статье.
3. **Изучайте** теорию и запускайте примеры кода в интерпретаторе Python.
4. **Переходите** по ссылкам «Далее» внизу каждой страницы для последовательного изучения.

Никакой сервер или сборка не требуются — сайт работает полностью на стороне клиента.

## 🎯 Цель проекта

Помочь начинающим программистам освоить фундаментальные концепции Python в логической последовательности:

- от базовых понятий (переменные, типы данных, ввод/вывод)
- через управляющие конструкции (условия, циклы)
- к структурам данных (строки, списки, кортежи, множества, словари)
- и функциональным возможностям (функции, списочные выражения)

## ✨ Особенности

- **Единый дизайн** — все страницы используют общий CSS-файл `style.css` с градиентными шапками, карточками-ссылками и единообразной типографикой.
- **Адаптивная сетка** — главная страница использует `grid-template-columns: repeat(auto-fill, minmax(280px, 1fr))` для корректного отображения на любых экранах.
- **Навигация между уроками** — внизу каждой страницы есть ссылки «Назад» и «Далее» для последовательного изучения.
- **Визуальные блоки** — заметки (`note`), советы (`tip`) и предупреждения (`warning`) выделены цветными боковыми панелями.
- **Подсветка кода** — все примеры Python-кода оформлены в тёмных блоках с моноширинным шрифтом.
- **Таблицы** — используются для наглядного представления справочной информации (операторы, методы, история версий).
