18. break

Прерывание цикла

Как работает break

break немедленно завершает текущий цикл:


for i in range(10):
    if i == 5:
        break
    print(i)  # 0 1 2 3 4

print("Цикл завершён")

Поиск с break

break удобен для поиска элемента:


numbers = [3, 7, 1, 9, 4, 2]
search = 9

for num in numbers:
    if num == search:
        print(f"Нашли {search}!")
        break
else:
    print(f"{search} не найден")

break в while

Часто используется для выхода из бесконечного цикла:


while True:
    cmd = input("> ")
    if cmd == "quit":
        break
    print(f"Выполняю: {cmd}")

break vs return

return завершает функцию целиком, break — только цикл:


def find_first_even(numbers):
    for num in numbers:
        if num % 2 == 0:
            return num  # return завершает функцию
    return None

for i in range(10):
    if i == 3:
        break  # break завершает только цикл
    print(i)
print("Продолжаем после цикла")
