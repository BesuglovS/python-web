16. range()

Генерация последовательностей чисел

Три формы range()

range(stop) — от 0 до stop-1:


list(range(5))     # [0, 1, 2, 3, 4]

range(start, stop) — от start до stop-1:


list(range(2, 7))  # [2, 3, 4, 5, 6]

range(start, stop, step) — с шагом step:


list(range(0, 10, 2))  # [0, 2, 4, 6, 8]
list(range(10, 0, -2)) # [10, 8, 6, 4, 2]

range() в цикле for

Самое частое применение — повторение N раз:


for i in range(3):
    print("Hello")  # Hello Hello Hello

# Или с индексом:
words = ["яблоко", "банан", "вишня"]
for i in range(len(words)):
    print(f"{i}: {words[i]}")

Особенности range()

range() не создаёт список, а генерирует числа на лету (экономит память):


# range — это отдельный тип
r = range(1000000)
print(type(r))  # 
print(len(r))   # 1000000
print(r[0])     # 0
print(r[-1])    # 999999

# Преобразовать в список при необходимости
nums = list(range(5))  # [0, 1, 2, 3, 4]
