20. continue

Пропуск итерации цикла

Как работает continue

continue пропускает оставшуюся часть текущей итерации и переходит к следующей:

for i in range(5):
    if i == 2:
        continue
    print(i)  # 0 1 3 4 (пропустили 2)

Пример: печать только чётных

Пропускаем нечётные числа:

for num in range(1, 11):
    if num % 2 != 0:
        continue
    print(num)  # 2 4 6 8 10

continue в while

В while важно не забыть обновить счётчик до continue:

num = 0
while num < 10:
    num += 1
    if num % 3 == 0:
        continue
    print(num)  # 1 2 4 5 7 8 10

break vs continue

Сравнение:

# break — выход из цикла
for i in range(5):
    if i == 3:
        break
    print(i)  # 0 1 2

# continue — пропуск итерации
for i in range(5):
    if i == 3:
        continue
    print(i)  # 0 1 2 4
