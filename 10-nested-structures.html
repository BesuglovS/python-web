10. Вложенные структуры

Условия внутри условий, вложенные блоки

Вложенные условные операторы

if может находиться внутри другого if:


age = 20
has_id = True

if age >= 18:
    print("Возраст подходит")
    if has_id:
        print("Вход разрешён")
    else:
        print("Нужно удостоверение личности")
else:
    print("Вход запрещён")

Глубокое вложение

Избегайте слишком глубокого вложения (более 3-4 уровней):


# Сложно читать:
if a:
    if b:
        if c:
            print("Все условия выполнены")

# Лучше:
if a and b and c:
    print("Все условия выполнены")

Вложенные циклы (анонс)

Циклы тоже могут быть вложенными (подробнее в теме 20):


for i in range(3):
    for j in range(2):
        print(f"i={i}, j={j}")
