21. Множества (set)

set(), операции над множествами

Создание множества

Множество — неупорядоченная коллекция уникальных элементов:


# Через фигурные скобки
fruits = {"яблоко", "банан", "вишня"}

# Через функцию set()
numbers = set([1, 2, 2, 3, 3, 3])  # {1, 2, 3}

# Пустое множество (не {} — это словарь!)
empty = set()

Операции над множествами

Математические операции:


a = {1, 2, 3, 4}
b = {3, 4, 5, 6}

print(a | b)   # {1, 2, 3, 4, 5, 6} — объединение
print(a & b)   # {3, 4} — пересечение
print(a - b)   # {1, 2} — разность
print(a ^ b)   # {1, 2, 5, 6} — симметрическая разность

Методы множеств

Основные методы:


s = {1, 2, 3}
s.add(4)          # {1, 2, 3, 4}
s.remove(2)       # {1, 3, 4} — ошибка, если нет
s.discard(10)     # {1, 3, 4} — без ошибки
s.pop()           # удаляет и возвращает случайный элемент
s.clear()         # set() — очищает множество

print(3 in s)     # True — проверка вхождения
