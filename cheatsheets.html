📖 1. Введение в Python

Что такое Python?

Интерпретируемый, высокоуровневый язык с динамической типизацией. Создан Гвидо ван Россумом в 1991 г. Назван в честь Monty Python.

Области применения

Веб (Django, Flask), Data Science (NumPy, Pandas), ML (TensorFlow), автоматизация, игры (PyGame), боты, DevOps.

⚠️ 9. Обработка ошибок (урок 27)

try / except / finally

    x = int(input())
except ValueError:
    print("Не число!")
except Exception as e:
    print(f"Ошибка: {e}")
else:
    print("OK:", x)
finally:
    print("Конец")

🏗️ 12. ООП (уроки 33, 33a)

Класс и объект

class Dog:
    def __init__(self, name):
        self.name = name

    def bark(self):

Наследование

class Puppy(Dog):
    def __init__(self, name, toy):
        super().__init__(name)
        self.toy = toy

    def bark(self):   # переопределение

def timer(fn):
    def wrapper(*a, **kw):
        start = time()
        res = fn(*a, **kw)
        print(time()-start)
        return res
    return wrapper
