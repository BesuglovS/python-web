22. Строки: индексация и срезы

Индексы, [start:stop:step]

Индексация строк

Каждый символ строки имеет индекс — номер позиции:

s = "Python"
#    0 1 2 3 4 5  — положительные индексы
#   -6-5-4-3-2-1  — отрицательные индексы

print(s[0])    # P
print(s[2])    # t
print(s[-1])   # n (последний символ)
print(s[-3])   # h (третий с конца)

Срезы строк [start:stop:step]

Срез — подстрока от start до stop-1 с шагом step:

s = "Python Programming"

print(s[0:6])       # "Python" — от 0 до 5
print(s[7:18])      # "Programming" — от 7 до 17
print(s[:6])        # "Python" — от начала
print(s[7:])        # "Programming" — до конца
print(s[::2])       # "Pto rgamn" — каждый второй
print(s[::-1])      # "gnimmargorP nohtyP" — разворот
print(s[-11:-1])    # "Programmin" — отрицательные индексы

Неизменяемость строк

Строки нельзя изменить — можно только создать новую:

s = "hello"
# s[0] = "H"  # Ошибка! TypeError

# Правильно — создать новую строку
s = "H" + s[1:]  # "Hello"

# len() — длина строки
print(len("Python"))  # 6
