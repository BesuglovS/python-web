2. Создание проекта в IDLE, VS Code и PyCharm

Настройка окружения для разработки на Python

Самый простой способ начать — использовать IDLE, встроенную среду, которая устанавливается вместе с Python. Для более серьёзной работы подойдут Visual Studio Code (VS Code) и PyCharm. Прежде чем писать код, нужно подготовить рабочее окружение. Современные IDE (Integrated Development Environment) — среды разработки — предоставляют удобный редактор кода, автодополнение, отладчик и управление проектами. В этом уроке мы рассмотрим все три инструмента.

Обзор IDLE, VS Code и PyCharm

Все IDE отлично подходят для изучения Python. Вы можете выбрать любую — разница для начинающего минимальна. Мы рассмотрим их чтобы вы могли принять взвешенное решение.

Запуск Python через IDLE (встроенная среда)

IDLE (Integrated Development and Learning Environment) — это простая среда разработки, которая входит в состав дистрибутива Python. Чтобы её использовать, необходимо сначала скачать и установить сам интерпретатор Python с официального сайта python.org. При установке обязательно отметьте галочку «Add Python to PATH» — это позволит запускать Python из командной строки. IDLE не требует дополнительной установки или настроек — идеальный вариант для первых шагов.

Как запустить IDLE

Способ зависит от вашей операционной системы:

  • Windows: нажмите Win, введите IDLE и выберите приложение «IDLE (Python 3.xx)». Либо откройте Пуск → Python 3.xx → IDLE.
  • macOS: откройте Terminal и выполните команду idle3 (или idle).
  • Linux: в терминале выполните idle3 или idle. Если команда не найдена, установите IDLE через пакетный менеджер: sudo apt install idle3 (Debian/Ubuntu) или sudo dnf install python3-idle (Fedora).

Создание и запуск файла в IDLE

  1. Запустите IDLE — откроется окно интерактивной оболочки (Python Shell).
  2. Нажмите Ctrl+N (или File → New File), чтобы создать новый файл.
  3. Напишите код, например: 
    print("Привет, IDLE!")
print("Python работает!")
  4. Сохраните файл (Ctrl+S или File → Save) с расширением .py, например main.py.
  5. Запустите программу клавишей F5 или командой Run → Run Module. Результат появится в окне Python Shell.

Работа в интерактивном режиме (Python Shell)

IDLE можно использовать как калькулятор или песочницу: после запуска сразу появляется приглашение >>>. Введите команду и нажмите Enter — результат покажется мгновенно: 

>>> 2 + 2
4
>>> print("Hello, World!")
Hello, World!

Совет: IDLE отлично подходит для самых первых экспериментов — в нём нет лишних настроек, он сразу готов к работе. Когда вы освоитесь, можно перейти к более мощным инструментам — VS Code или PyCharm.

Настройка VS Code

Шаг 1. Установка VS Code

Скачайте установщик с официального сайта code.visualstudio.com. Запустите установку — все параметры можно оставить по умолчанию.

Шаг 2. Установка Python

Скачайте Python с python.org. При установке обязательно отметьте галочку «Add Python to PATH» — это позволит запускать Python из командной строки.

Проверьте установку в терминале:

python --version
# Пример вывода: Python 3.13.x

Шаг 3. Установка расширения Python

VS Code сам по себе не знает Python — ему нужно расширение. Нажмите Ctrl+Shift+X (или иконка расширений слева), в поиске введите «Python» и установите расширение от Microsoft: 

# Расширение: Python (by Microsoft)
# ID: ms-python.python

Это расширение добавляет подсветку синтаксиса, автодополнение (IntelliSense), отладчик, форматирование кода и поддержку Jupyter Notebook.

Шаг 4. Создание проекта

Создайте папку для проекта и откройте её в VS Code:

  1. Нажмите Ctrl+K Ctrl+O (или File → Open Folder)
  2. Выберите или создайте новую папку, например my-first-project
  3. Нажмите Ctrl+N — создайте новый файл
  4. Сохраните его с расширением .py, например main.py

Шаг 5. Запуск кода

Напишите простую программу и запустите её:

print("Привет, VS Code!")
print("Python работает!")

Способы запуска:

  • Через терминал: Ctrl+Shift+` → введите python main.py
  • Кнопка Run ▶ (в правом верхнем углу)
  • Правой кнопкой мыши на файле → «Run Python File in Terminal»

Совет: Установите также расширение Python Debugger (ms-python.debugger) для удобной отладки кода с точками останова.

Дополнительные полезные расширения

  • Pylance — улучшенное автодополнение и проверка типов
  • Python Docstring Generator — автоматическая генерация docstring
  • Jupyter — поддержка Jupyter Notebook прямо в VS Code
  • GitLens — удобная работа с Git
  • Prettier / Black — форматирование кода

Настройка PyCharm

Шаг 1. Установка PyCharm

Скачайте установщик с официального сайта jetbrains.com/pycharm. Для обучения достаточно PyCharm Community Edition — он бесплатен и содержит все необходимые функции.

Шаг 2. Установка Python

Если вы ещё не установили Python — сделайте это так же, как описано в разделе VS Code. PyCharm может автоматически обнаружить установленный Python при создании проекта.

Шаг 3. Создание нового проекта

Запустите PyCharm — откроется окно приветствия:

  1. Нажмите «New Project»
  2. Выберите расположение проекта (Location), например C:\Users\Имя\PycharmProjects\my-first-project
  3. В разделе «Base interpreter» выберите установленный Python (PyCharm обычно находит его автоматически)
  4. Отметьте «Create a main.py welcome script» — будет создан файл main.py с готовым кодом
  5. Нажмите «Create»

Шаг 4. Запуск кода

PyCharm создаст файл main.py с шаблоном. Вы можете заменить его содержимое: 

print("Привет, PyCharm!")
print("Python работает!")

Способы запуска:

  • Зелёная стрелка ▶ в правом верхнем углу
  • Правой кнопкой мыши на файле → «Run 'main'»
  • Горячие клавиши: Shift+F10

Важно: PyCharm может автоматически создавать виртуальное окружение (venv) для каждого проекта — это хорошая практика, так как зависимости разных проектов не будут конфликтовать.

Полезные возможности PyCharm

  • Автодополнение кода — PyCharm предлагает варианты по мере ввода
  • Отладчик — ставьте точки останова (кликнув слева от номера строки) и нажимайте иконку жука для отладки
  • Рефакторинг — переименование переменных, выделение методов и т.д.
  • Проверка кода — PyCharm подсвечивает ошибки и предлагает исправления
  • Интеграция с Git — встроенные инструменты для работы с системами контроля версий
  • Навигация — быстрый переход к определению функции или класса (Ctrl+Click)

Советы по настройке PyCharm

  • Тема оформления: File → Settings → Appearance & Behavior → Theme (выберите Dracula, One Dark или другую)
  • Размер шрифта: Settings → Editor → Font
  • Автосохранение: PyCharm сохраняет файлы автоматически, но можно включить явно в Settings → Appearance & Behavior → System Settings → Synchronization

Python "из коробки" Да Нет (нужно расширение) Да

Что выбрать? Если вы делаете самые первые шаги — начните с IDLE, он не требует установки и настроек. Если хотите что-то лёгкое и универсальное — переходите на VS Code. Если планируете серьёзно изучать Python и Data Science — PyCharm Community Edition будет отличным выбором. Все три среды бесплатны.

Совет: Попробуйте все три! Начните с IDLE, затем установите VS Code и PyCharm, создайте в каждом простой проект и выберите ту среду, в которой вам комфортнее работать.

Виртуальное окружение (venv)

Виртуальное окружение — это изолированная среда для Python-проекта, в которой устанавливаются свои пакеты и зависимости. Это нужно, чтобы библиотеки разных проектов не конфликтовали между собой.

Создание виртуального окружения в терминале:

# Создать виртуальное окружение
python -m venv .venv

# Активировать (Windows)
.venv\Scripts\activate

# Активировать (Mac/Linux)
source .venv/bin/activate

# Установить пакет
pip install numpy

# Выйти из окружения
deactivate

VS Code предложит использовать виртуальное окружение при открытии проекта. PyCharm создаёт его автоматически при создании нового проекта.
